# 🎓 EduPath AI
### Your Personal AI Career & College Guidance Platform for Engineering Aspirants

## 🚀 Problem Statement
Every year, millions of students struggle to:
- Choose the right engineering college
- Select the best branch
- Understand placement opportunities
- Plan their career roadmap
- Find personalized guidance

EduPath AI solves these challenges using Artificial Intelligence, Multi-Agent Systems, and Personalized Recommendations.

---

## ✨ Features

### 🎯 College Predictor
- Predict colleges based on percentile/rank
- Category-wise recommendations
- Safe, Target, and Reach analysis

### 🤖 AI Career Counselor
- Personalized branch recommendations
- Future scope and salary insights
- Industry demand analysis

### 📊 Placement Dashboard
- Average and highest packages
- ROI comparison
- Top recruiters

### 🗺️ Study Roadmap Generator
- Personalized 4-year roadmap
- Projects and certifications
- Skill recommendations

### 📚 Smart PDF Tutor
- PDF summaries
- MCQs and quizzes
- Explanations and notes

### 🧠 Personalized Memory
- Saves preferences
- Search history
- Customized recommendations

## ⚙️ Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
